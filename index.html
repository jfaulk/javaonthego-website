What is Java on the Go!

Java on the go is a mobile educational application focused on providing a way for beginners to easily get a grasp of the basics of the Java programming language.

Start Now!

Why

Mobile devices are an essential part of living in the modern, connected world. Java On The Go makes learning easy by always having the tools you need within hands reach.


About Java On The Go

Java On The Go is a labour of love created by Ryan, Jeremy, James and Jonathan, a group of students studying at George Brown College. They were disappointed with the current products available on the market and wanted to deliver the best way to learn Java on your mobile device.
